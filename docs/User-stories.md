# User Stories - TOEIC Coach AI MVP

## 1. Authentication & Profile

### US-01 - Register
As a Student,  
I want to create an account,  
so that I can save my learning progress.

### US-02 - Login
As a Student,  
I want to log in to my account,  
so that I can continue learning from my previous progress.

### US-03 - Update Learning Goal
As a Student,  
I want to set my target TOEIC score and exam date,  
so that the system can personalize my roadmap.

### US-04 - Update Study Availability
As a Student,  
I want to set how many minutes I can study per day,  
so that the roadmap matches my real schedule.

---

## 2. Placement Test

### US-05 - Take Placement Test
As a Student,  
I want to take a placement test,  
so that I can know my current TOEIC level.

### US-06 - View Estimated Score
As a Student,  
I want to view my estimated TOEIC score after the placement test,  
so that I know my starting point.

### US-07 - View Initial Weakness
As a Student,  
I want to see my weak TOEIC Parts and topics,  
so that I know what to improve first.

---

## 3. Question Bank

### US-08 - View Practice Questions
As a Student,  
I want to access TOEIC practice questions,  
so that I can improve my skills.

### US-09 - Practice by TOEIC Part
As a Student,  
I want to practice by TOEIC Part,  
so that I can focus on a specific exam section.

### US-10 - Practice by Grammar Topic
As a Student,  
I want to practice by grammar topic,  
so that I can improve weak grammar areas.

### US-11 - Practice by Business Topic
As a Student,  
I want to practice by business topic,  
so that I can learn TOEIC vocabulary in real contexts.

### US-12 - Practice by Difficulty
As a Student,  
I want to practice questions from easy to hard,  
so that I can improve step by step.

---

## 4. Listening Audio

### US-13 - Play Audio Question
As a Student,  
I want to listen to audio for Listening questions,  
so that I can practice TOEIC Listening.

### US-14 - View Transcript After Submission
As a Student,  
I want to view the transcript after submitting my answer,  
so that I can understand what I missed.

---

## 5. Submission & Result

### US-15 - Submit Practice Session
As a Student,  
I want to submit my answers,  
so that the system can calculate my result.

### US-16 - View Score
As a Student,  
I want to view my score after a practice session,  
so that I know how well I performed.

### US-17 - Review Wrong Answers
As a Student,  
I want to review my wrong answers,  
so that I can learn from my mistakes.

### US-18 - Save Learning History
As a Student,  
I want the system to save my practice history,  
so that I can track my progress over time.

---

## 6. Weakness Analysis

### US-19 - View Weak Topics
As a Student,  
I want to see my weakest grammar and business topics,  
so that I know what to focus on next.

### US-20 - View Accuracy by Part
As a Student,  
I want to see my accuracy by TOEIC Part,  
so that I know which sections need improvement.

### US-21 - View Accuracy by Difficulty
As a Student,  
I want to see my accuracy by difficulty level,  
so that I know whether I am ready for harder questions.

---

## 7. Roadmap Generator

### US-22 - Generate Roadmap
As a Student,  
I want the system to generate a learning roadmap,  
so that I know what to study each week.

### US-23 - View Weekly Plan
As a Student,  
I want to view my weekly study plan,  
so that I can follow a clear learning path.

### US-24 - Update Roadmap After Practice
As a Student,  
I want the roadmap to update based on my performance,  
so that it stays relevant to my weaknesses.

---

## 8. Dashboard

### US-25 - View Dashboard
As a Student,  
I want to view my learning dashboard,  
so that I can quickly understand my progress.

### US-26 - View Progress Toward Target
As a Student,  
I want to compare my current score with my target score,  
so that I know how far I still need to go.

### US-27 - View Recent Activities
As a Student,  
I want to see my recent practice sessions,  
so that I can review my learning activity.

---

## 9. Flashcards

### US-28 - Study Flashcards by Topic
As a Student,  
I want to study flashcards by topic,  
so that I can improve TOEIC vocabulary.

### US-29 - Review Flashcard
As a Student,  
I want to mark a flashcard as Easy, Medium, or Hard,  
so that the system can schedule the next review.

### US-30 - View Due Flashcards
As a Student,  
I want to see flashcards due for review today,  
so that I can review vocabulary at the right time.

---

## 10. AI Answer Explanation

### US-31 - Request AI Explanation
As a Student,  
I want to ask AI to explain a question,  
so that I can understand why the correct answer is right.

### US-32 - Explain Wrong Answer
As a Student,  
I want AI to explain why my selected answer was wrong,  
so that I can avoid the same mistake next time.

### US-33 - Get Quick TOEIC Tip
As a Student,  
I want AI to provide a short TOEIC tip for a question,  
so that I can learn faster test-taking strategies.

---

## 11. AI Learning Coach

### US-34 - Get AI Learning Feedback
As a Student,  
I want AI to summarize my learning performance,  
so that I know my strengths and weaknesses.

### US-35 - Get Next Study Suggestion
As a Student,  
I want AI to suggest what I should study next,  
so that I can focus on the most important improvement area.

---

## 12. Admin - Question Management

### US-36 - Create Question
As an Admin,  
I want to create TOEIC questions,  
so that students can practice with new content.

### US-37 - Update Question
As an Admin,  
I want to update existing questions,  
so that I can fix mistakes or improve content quality.

### US-38 - Delete Question
As an Admin,  
I want to delete invalid questions,  
so that students do not practice incorrect content.

### US-39 - Import Questions
As an Admin,  
I want to import questions from CSV or JSON,  
so that I can add many questions quickly.

### US-40 - Upload Audio
As an Admin,  
I want to upload audio files for Listening questions,  
so that students can practice Part 1-4.

---

## 13. Admin - User Management

### US-41 - View Users
As an Admin,  
I want to view registered users,  
so that I can manage the platform.

### US-42 - Disable User
As an Admin,  
I want to disable a user account,  
so that I can prevent abuse or invalid usage.

---
